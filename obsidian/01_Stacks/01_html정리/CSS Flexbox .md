# 🎨 CSS Flexbox 속성 완벽 정리

CSS Flexbox 레이아웃을 구성하는 핵심 속성들을 부모(Container)와 자식(Item) 요소로 나누어 정리한 노트입니다.

## ⚡️ 한눈에 보는 요약표

| 속성                  | 적용 대상 | 역할      | 비유 (이해 쏙쏙)          |
| :------------------ | :---- | :------ | :------------------ |
| **flex-direction**  | 부모    | 나열 방향   | 가로본능 vs 세로본능        |
| **flex-wrap**       | 부모    | 줄바꿈 여부  | 꾸역꾸역 넣기 vs 넘치면 다음 줄 |
| **justify-content** | 부모    | 주축 정렬   | (가로) 좌/우/중앙/간격벌리기   |
| **align-items**     | 부모    | 교차축 정렬  | (세로) 위/아래/중앙 (한 줄)  |
| **align-content**   | 부모    | 여러 줄 정렬 | (세로) 뭉텅이 전체 정렬      |
| **order**           | 자식    | 순서 변경   | 새치기                 |
| **align-self**      | 자식    | 개별 정렬   | 독불장군 (나만 다르게)       |

---

## 1. 부모 요소(Container) 속성: 판을 짜는 역할
> 아이템들이 담겨 있는 전체 박스에 적용하는 속성들입니다.

### 📌 `flex-direction` (방향 설정)
아이템들이 나열되는 **주축(Main Axis)** 의 방향을 결정합니다.
* **`row` (기본값):** 왼쪽에서 오른쪽으로 (가로)
* **`column`:** 위에서 아래로 (세로)
* **`row-reverse` / `column-reverse`:** 반대 방향으로 나열

### 📌 `flex-wrap` (줄 바꿈 설정)
아이템들이 한 줄에 다 들어가지 않을 때, 줄을 바꿀지 결정합니다.
* **`nowrap` (기본값):** 줄 바꿈 없이 억지로 한 줄에 우겨 넣습니다. (사이즈가 줄어듦)
* **`wrap`:** 공간이 부족하면 다음 줄로 넘어갑니다.

### 📌 `flex-flow` (단축 속성)
위의 두 가지(`flex-direction` + `flex-wrap`)를 한 번에 적는 속성입니다.
* 예시: `flex-flow: row wrap;` (가로로 나열하되, 꽉 차면 줄 바꿈)

---

## 2. 정렬 관련 속성 (부모 요소 적용)
> 아이템들을 예쁘게 배치할 때 가장 많이 사용됩니다.

### 📐 `justify-content` (주축 정렬)
`flex-direction`이 가로(`row`)라면 **가로 정렬**, 세로(`column`)라면 **세로 정렬**을 담당합니다.
* **`flex-start`:** 시작점 정렬 (왼쪽)
* **`center`:** 가운데 정렬
* **`space-between`:** 양 끝에 붙이고 아이템 간 간격을 균등하게
* **`space-around`:** 아이템 양옆에 균등한 여백을 줌

### 📐 `align-items` (교차축 정렬)
주축과 반대되는 **교차축(Cross Axis)** 정렬입니다. 보통 세로 정렬을 맞출 때 씁니다. (**한 줄 기준**)
* **`stretch` (기본값):** 아이템을 부모 높이만큼 쭈욱 늘림
* **`center`:** 교차축의 중앙에 배치
* **`flex-start` / `flex-end`:** 위쪽 / 아래쪽 정렬

### 📐 `align-content` (여러 줄 정렬)
**두 줄 이상(`flex-wrap: wrap`)일 때만 의미가 있습니다.** 여러 줄 뭉텅이 전체를 교차축 방향으로 어떻게 정렬할지 결정합니다.
* `align-items`와 값은 비슷하지만, **한 줄일 때는 동작하지 않습니다.**

---

## 3. 자식 요소(Item) 속성: 개별 제어
> 특정 아이템 하나만 다르게 하고 싶을 때 자식 요소에 직접 적용합니다.

### 🔹 `order` (순서 변경)
HTML 코드의 순서를 바꾸지 않고, **화면에 보이는 순서만** 바꿉니다.
* 기본값은 `0`입니다.
* 숫자가 작을수록 앞에, 클수록 뒤에 배치됩니다.
* 예시: `order: -1;` (맨 앞으로 이동), `order: 5;` (뒤로 이동)

### 🔹 `align-self` (나홀로 정렬)
부모의 `align-items` 설정을 무시하고, **이 아이템 하나만 따로 정렬**하고 싶을 때 씁니다.
* 예시: 다른 애들은 다 가운데(`center`)인데, 나만 위(`flex-start`)에 붙고 싶을 때 사용.
