# [HTML] 01_Structure: 문서 기본 구조 및 설정

> [!info] **학습 목표** 웹 페이지의 뼈대를 구성하는 기본 태그들의 역할을 이해하고, 브라우저와 검색 엔진에 필요한 정보를 제공하는 방법을 배웁니다.

---

## 1. HTML 기본 보일러플레이트 (Boilerplate)

보일러플레이트란 '항상 똑같이 들어가는 기본 틀'을 의미합니다. 집을 지을 때 기초 공사를 하는 것과 같습니다.

HTML

```
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>페이지 제목</title>
</head>
<body>
    <h1>안녕하세요! 화면에 보이는 내용입니다.</h1>
</body>
</html>
```

- **`<!DOCTYPE html>`**: "이 문서는 최신 HTML5 표준으로 작성되었습니다"라고 브라우저에게 알리는 선언문입니다.
    
- **`<html>`**: 전체 문서의 시작과 끝을 감싸는 뿌리(Root) 태그입니다. `lang="ko"` 속성은 이 페이지가 한국어 위주임을 알려주어 검색 엔진과 스크린 리더(시각장애인용 도구)를 돕습니다.
    
- **`<head>`**: **"머리속 정보"**. 브라우저 화면에는 보이지 않지만, 문서의 설정, 제목, 외부 파일 연결 등 중요한 데이터를 담습니다.
    
- **`<body>`**: **"몸통(실제 화면)"**. 사용자의 눈에 보이는 모든 콘텐츠(글, 그림, 버튼 등)가 들어가는 곳입니다.
    

---

## 2. Meta 태그와 SEO (Search Engine Optimization)

메타 태그는 "데이터에 대한 데이터"입니다. 내 사이트가 검색 엔진(구글, 네이버)에 어떻게 보일지, 기기마다 어떻게 보여질지 결정합니다.

- **`<meta charset="UTF-8">`**: 한글을 포함한 전 세계 모든 언어를 깨짐 없이 표시하기 위한 문자 인코딩 설정입니다. (안 쓰면 한글이 외계어로 보일 수 있어요!)
    
- **`<meta name="viewport" content="width=device-width, initial-scale=1.0">`**: **반응형 웹의 핵심**입니다. 기기(모바일, 태블릿, PC)의 너비에 맞춰 화면 크기를 자동으로 조절해 줍니다.
    
- **`<meta name="description" content="페이지 설명">`**: 검색 결과 창 제목 아래에 나오는 짧은 설명글입니다. 클릭을 유도하는 중요한 SEO 요소입니다.

## 3. Link 태그와 Favicon

외부에 있는 파일을 불러오거나 사이트의 정체성을 보여주는 아이콘을 설정할 때 사용합니다.

- **`<link rel="stylesheet" href="style.css">`**: CSS 파일(디자인)을 연결할 때 사용합니다. `head` 태그 안에 위치합니다.
    
- **Favicon (파비콘)**: 브라우저 탭 왼쪽 상단에 보이는 작은 아이콘입니다.
    
    HTML
    
    ```
    <link rel="icon" href="favicon.ico" type="image/x-icon">
    ```
