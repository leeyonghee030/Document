# Java 기초 메모  
  
## Java가 뭐야?  
  
Java는 프로그램을 만들기 위한 **프로그래밍 언어**이다.    
웹, 앱, 서버 프로그램 등 여러 곳에서 사용된다.  
  
---  
  
## 1. Java 프로그램의 기본 단위: 클래스  
  
Java 프로그램은 보통 **클래스(Class)** 를 중심으로 작성한다.  
  
- 클래스는 쉽게 말해 **설계도**이다.  
- 비슷한 성질을 가진 객체들을 만들기 위한 틀이다.  
- 클래스 안에는 **데이터**와 **기능**이 들어간다.  
  
### 클래스 안에 들어가는 것  
- **데이터**: 변수(필드)  
- **기능**: 메소드(Method)  
  
예를 들어,  
- `Car`라는 클래스가 있으면  
- 데이터: 색깔, 속도  
- 기능: 달리기, 멈추기  
  
처럼 만들 수 있다.  
  
---  
  
## 2. 객체란?  
  
객체(Object)는 **클래스를 바탕으로 실제로 만들어진 것**이다.  
  
- 클래스 = 설계도  
- 객체 = 설계도로 만든 실제 물건  
  
예를 들어,  
- 클래스: `Car`  
- 객체: 실제 자동차 1대  
  
즉,    
**객체는 특정 클래스의 인스턴스(Instance)** 라고 한다.

## 3. Java는 자료형 검사가 엄격하다  
  
Java는 변수의 **자료형(Type)** 을 중요하게 생각한다.  
  
예를 들어,  
- 숫자를 저장하는 변수  
- 글자를 저장하는 변수  
- 참/거짓을 저장하는 변수  
  
이렇게 용도에 맞게 나누어 사용해야 한다.  
  
Java는 자료형이 맞지 않으면 오류가 나기 때문에    
초보자에게는 조금 엄격하게 느껴질 수 있다.  
  
### 예시  
```java  
int age = 20;  
String name = "Kim";

- `int` : 정수
    
- `String` : 문자열

## 4. Java를 공부하거나 개발하려면 JDK가 필요하다

Java 프로그램을 만들고 실행하려면 **JDK**를 설치해야 한다.

### JDK란?

**JDK(Java Development Kit)** 는  
Java 프로그램을 개발하는 데 필요한 도구 모음이다.

JDK 안에는 보통 이런 것들이 들어 있다.

- 자바 코드를 컴파일하는 도구
    
- 자바 프로그램을 실행하는 도구
    
- 기본 라이브러리
    

즉,  
**Java 개발에 필요한 기본 세트**라고 보면 된다.

---

## 5. Java 소스가 실행되는 과정

Java 프로그램은 바로 실행되지 않고, 다음 과정을 거친다.

### 1단계: 소스 코드 작성

사람이 읽을 수 있는 Java 코드를 작성한다.

예:

public class Hello {  
    public static void main(String[] args) {  
        System.out.println("Hello Java");  
    }  
}

### 2단계: 컴파일

작성한 Java 소스를 컴파일하면 **바이트코드(Bytecode)** 로 바뀐다.

- 소스 코드: 사람이 읽기 쉬운 코드
    
- 바이트코드: 컴퓨터가 실행하기 좋은 중간 형태의 코드
    

### 3단계: 실행

이 바이트코드는 **Java 플랫폼** 위에서 실행된다.

---

## 6. Java 플랫폼이란?

Java 플랫폼은 보통 다음 두 가지로 이루어진다고 본다.

- **Java VM(JVM)**
    
- **Java API**
    

### JVM이란?

**JVM(Java Virtual Machine)** 은  
바이트코드를 읽어서 실행해 주는 가상 머신이다.

쉽게 말하면,  
**Java 프로그램을 실제로 돌려주는 실행기**이다.

### Java API란?

Java에서 기본으로 제공하는 여러 기능들의 모음이다.

예를 들어,

- 문자열 처리
    
- 날짜 처리
    
- 파일 읽기
    
- 리스트 사용
    

같은 기능을 Java API를 통해 사용할 수 있다.

## 7. main() 메소드란?

Java 프로그램은 보통 **main() 메소드**에서 시작한다.

형식은 다음과 같다.

public static void main(String[] args)

이 형식은 Java 프로그램의 시작점으로 자주 사용된다.

### 각 부분의 뜻

- `public`  
    → 다른 곳에서도 접근할 수 있다.
    
- `static`  
    → 객체를 만들지 않아도 실행할 수 있다.
    
- `void`  
    → 실행 후 돌려주는 값이 없다.
    
- `main`  
    → 프로그램이 시작되는 메소드 이름이다.
    
- `String[] args`  
    → 실행할 때 전달받는 문자열 값들이다.

## 8. 핵심 한 번에 정리

- **클래스**: 객체를 만들기 위한 설계도
    
- **객체**: 클래스를 바탕으로 실제로 만들어진 것
    
- **인스턴스**: 특정 클래스로부터 생성된 객체
    
- **자료형**: 변수에 어떤 종류의 값을 넣을지 정하는 것
    
- **JDK**: Java 개발에 필요한 도구 모음
    
- **컴파일**: 소스 코드를 바이트코드로 바꾸는 과정
    
- **바이트코드**: JVM이 실행할 수 있는 중간 코드
    
- **JVM**: 바이트코드를 실행하는 가상 머신
    
- **Java API**: Java에서 제공하는 기본 기능 모음
    
- **main()**: 프로그램이 시작되는 메소드
