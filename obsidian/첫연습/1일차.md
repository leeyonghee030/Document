
## 💻 Git 작업 항목

### 📸 스크립샷 커밋

* `git add` 명령어로 스크린샷 파일을 스테이징 영역에 추가
* `git commit` 명령어로 커밋 메시지와 함께 로컬 저장소에 기록

### 💻 로컬 컴퓨터 체크아웃

* Git에서 **특정 상태** (커밋, 브랜치, 태그)로 로컬 작업 폴더를 전환하는 작업.

### 📤 GitHub 푸쉬 및 로컬 폴더로 가져오기

* **깃허브 푸쉬**: 로컬 저장소의 변경 사항을 원격 저장소(GitHub)로 업로드
* **깃허브를 내 로컬 폴더**: `git clone` (최초) 또는 `git pull` (업데이트) 명령어를 사용하여 원격 저장소 내용을 로컬로 가져옴

## 🌐 HTML(HyperText Markup Language)

### ❓ HTML 이란?

* **웹사이트**를 위해 만든 **마크업 언어**
* **하이퍼링크** 기능 포함

### 🏗️ HTML 구성 요소

| 구분 | 내용 |
| :--- | :--- |
| **문서** | **요소**로 구성 |
| **요소** | **태그 한 쌍** (+속성) (+중첩 요소 가능) |
| **태그** | 컨텐츠가 없는 태그도 있음 (이미지, 링크 등) |

### 🧭 HTML 구조

| 요소 | 역할 | 상세 내용 |
| :--- | :--- | :--- |
| **최상위 Root 요소** | `<html>` | `lang=en` (언어 약자) 속성 지정 |
| **헤드** (`<head>`) | 웹사이트에 대한 설명 **(메타데이터)**, 웹사이트 상에 **보여지지 않는 정보**들 (책 표지 역할) | SEO, 타이틀, 설명, 저자, CSS 스타일/JS 링크 |
| **바디** (`<body>`) | **웹사이트 내용** | `<ul>` 등 실제 컨텐츠 |

### 🏷️ `<head>` 태그 상세 (메타데이터)

* **`<meta>` 태그**를 사용하여 다양한 속성 지정 가능.
* **`charset` (문자열 셋)**:
    * `<meta charset="utf-8">`
    * `utf-8` 인코딩 사용 (다양한 언어를 깨짐 현상 없이 나타냄)
* `<meta http-equiv>`
* `<meta name>`
* **`<title>이름</title>`**
    * (여기까지가 헤드 영역)
